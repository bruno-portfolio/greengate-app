Como usar o GreenGate

1

Navegue e Defina a Área

Use a busca, desenhe com o Lápis, faça upload ou teste com área de exemplo. Limite: 10.000 ha.

2

Validação Automática

Clique em Validar Área para cruzamento instantâneo com 13 bases oficiais (PRODES, DETER, MapBiomas, TIs, Embargos, etc).

3

Relatório PDF

Baixe o relatório profissional com mapas, verificações detalhadas e QR Code de autenticidade. Requer API Key.
